# Button

Secondary buttons is used for secondary call to action on a given page and there should only be one large secondary call to action per page. Inactive button state appears when the customer has to do something obligatory to activate the button (e.g. when filling in mandatory sections of certain forms).

## How to use

### Installation

```bash
npm install --save @demo-ds/alc-button-secondary
yarn add @demo-ds/alc-button-secondary
```

```js
import '@demo-ds/alc-button-secondary';
```
